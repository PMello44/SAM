# SAM

One page portfolio site for a filmmaker and video editor.

## Estrutura

- `index.html`: estrutura principal da página
- `styles.css`: estilos e responsividade
- `darkveil.js`: fundo animado da hero
- `pixel-snow.js`: camada global de partículas
- `Assets/`: fontes e recursos locais

## Publicação no GitHub Pages

Este projeto é estático e pode ser publicado diretamente pelo GitHub Pages usando a branch `main`.
